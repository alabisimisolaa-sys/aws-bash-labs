# AWS Bash Scripting Labs

Hands-on bash scripting exercises completed as part of AWS Cloud Practitioner study program.

## Lab 1 — Dynamic File Creation Script

Creates 25 empty files per run with auto-incrementing names.

Each run detects the highest existing file number and continues from there — no hardcoded values.

**Skills:** Bash scripting, EC2 SSH, Linux file system, automation logic

**Environment:** Amazon Linux EC2 instance via AWS Management Console
